# Summary

This directory contains recordings of conversations about working on Microsoft Hyper-V.

# Common Terms to help with mistranscriptions

NUMA Node
Virt stack - Hyper-V virtualization stack
vmms - virtual machine management service (vmms.exe)
vmwp - virtual machine worker process (vmwp.exe)
vmcompute - virtual machine compute (vmcompute.exe)
VM-PHU - virtual machine preserving host updates
VMMP - virtual machine memory partition, synonym for the primary partition (which is a partition for primary guest ram)
VSMMP - virt stack metadata memory partition
HMMP - hypervisor metadata memory partition
VID - virtualization infrastructure driver
VP - virtual processor
Self - "Self" variant of VM-PHU
Lite - "Lite" variant of VM-PHU
Ultralite - "Ultralite" variant of VM-PHU
PF Agent
RD Agent
Kernel Agent
ICM - Incident Management (sev 1/2/3/etc)
Underhill
RI - Reverse Integration
FI - Forward Integration
IO Space
WHEA - Windows Hardware Error Architecture
CHIE - A Windows team
LOW - Linux On Windows team
BPT - Base Platform Team
SAW - Secure Access Workstation (a locked down laptop that can access Azure)
HHR - Hypervisor Hot Restart (technology to load a new hypervisor into memory and switch over to it with zero downtime)
HSR - Hypervisor Soft Restart
KSR - Kernel Soft Reboot
1.86 - Redstone verison 1.86
